# 신규회선 체크

명의자별 신규회선 가능일을 관리하는 작은 PWA 웹앱입니다.

## 주요 기능

- 명의자, 통신사, 신규날짜, 신규번호 입력
- 신규날짜 기준 다음 신규회선 가능일 자동 계산
- 회선 등록 후 다음 접속 시 목록 우선 표시
- 명의자별 요약과 펼치기/접기
- 브라우저 자동 저장
- JSON 백업 저장 / 백업 불러오기
- CSV 저장
- 검색, D-Day 표시, 메모, 중복 번호 경고
- PIN 잠금
- 스마트폰 홈 화면 추가용 PWA 구성

## 사용

컴퓨터에서는 `index.html`을 브라우저로 열면 바로 사용할 수 있습니다.

GitHub Pages 같은 HTTPS 주소에 올리면 스마트폰 브라우저에서 앱처럼 설치할 수 있습니다.

## GitHub Pages 배포

1. GitHub에서 새 저장소를 만듭니다.
2. 이 폴더의 파일을 저장소에 올립니다.
3. 저장소의 `Settings > Pages`에서 `Deploy from a branch`를 선택합니다.
4. Branch는 `main`, folder는 `/root`로 설정하고 저장합니다.
5. 잠시 후 생성되는 Pages 주소를 스마트폰에서 엽니다.
6. 브라우저 메뉴에서 `홈 화면에 추가`를 선택합니다.

## 데이터

입력한 데이터는 현재 브라우저의 로컬 저장소에 저장됩니다. 기기를 바꾸거나 데이터를 옮길 때는 `백업 저장`으로 만든 JSON 파일을 사용하세요.
