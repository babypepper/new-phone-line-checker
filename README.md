# 신규회선 체크

명의자별 신규회선 가능일을 관리하는 작은 PWA 웹앱입니다.

## 문구 수정과 업로드

제목 아래에 타이핑되는 문구와 공지 내용은 `site-content.json`에서 관리합니다.

1. `site-content.json`을 열어 문구를 수정합니다.
2. PowerShell에서 아래 명령을 실행합니다.

```powershell
.\publish-content.ps1
```

이 스크립트가 GitHub 최신 내용 가져오기, 단일 HTML 갱신, 커밋, 업로드까지 한 번에 처리합니다.

## 주요 기능

- 명의자, 통신사, 신규날짜, 신규번호 입력
- 신규날짜 기준 다음 신규회선 가능일 자동 계산
- 회선 등록 후 다음 접속 시 목록 우선 표시
- 명의자별 요약과 회선 목록 바로 이동
- 브라우저 자동 저장
- JSON 백업 저장 / 백업 불러오기
- D-Day 표시, 중복 번호 경고

## 데이터

입력한 데이터는 현재 브라우저의 로컬 저장소에 저장됩니다. 기기를 바꾸거나 데이터를 옮길 때는 `백업 저장`으로 만든 JSON 파일을 사용하세요.
